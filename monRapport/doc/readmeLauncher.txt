--------------------------Doc Launcher--------------------------


----------Specs----------

Ce fichier ouvre tous les fichiers dont les noms sont specifier dans avec l'option "changer les noms"; il charge les macros dans le fichier "extract"; puis il lance toutes les macros, les unes apres les autres.


----------Contraintes----------

Tous les fichiers et les macros doivent se trouver dans le meme dossier que le launcher.
